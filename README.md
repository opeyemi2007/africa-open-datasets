# 🌍 Africa Open Datasets

A free and open collection of structured datasets about African countries, cities, universities, languages, and resources.

Built for developers, researchers, students, and creators.

## 📚 Available Datasets

### 🇳🇬 Nigeria

- States
- Cities
- Universities
- Banks
- Languages

### 🇬🇭 Ghana

- Regions

### 🇰🇪 Kenya

- Counties
